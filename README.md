# DHguides — фиксы и материалы для Dread Hunger

## 🎙️ Фикс войса (голос пропадает после реконнекта)

Папка [`fixes/`](fixes):

| Файл | Для какой версии |
|------|------------------|
| `DH-VoiceFix-Steam.zip` | лицензия **Steam** (ставит UE4SS) |
| `DH-VoiceFix-FreeTP.zip` | **FreeTP / пиратская** сборка |

### Установка
1. Скачай нужный архив и распакуй.
2. Двойной клик по **`УСТАНОВИТЬ.vbs`**.
3. Если спросит — укажи папку **`Win64`** (где лежит `DreadHunger.exe`).
4. Запусти игру.
5. Если голос пропал после реконнекта — нажми **CTRL + M**.
